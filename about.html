📚 Про цей проект
Веб-сайт "CyberSecure" створений як практична робота для демонстрації навичок веб-розробки та розуміння принципів подання інформації з кібербезпеки. Мета проекту — створити інформативний та зручний ресурс, який допоможе користувачам ознайомитися з основами захисту в цифровому середовищі. Під час розробки були застосовані знання HTML, CSS та JavaScript, а також принципи адаптивного дизайну для коректного відображення на різних пристроях. Особлива увага приділялася структуруванню інформації та створенню інтуїтивно зрозумілого інтерфейсу.
