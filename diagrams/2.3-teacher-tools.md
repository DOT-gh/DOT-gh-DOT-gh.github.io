## Діаграма 2.3.2: Процес моніторингу учнів

\`\`\`mermaid
sequenceDiagram
    participant T as Вчитель
    participant Dashboard as Панель вчителя
    participant Monitor as Система моніторингу
    participant DB as База даних
    participant AI as AI-система
    participant S as Учень
    
    T->>Dashboard: Відкриває моніторинг
    Dashboard->>Monitor: Запит даних
    Monitor->>DB: Отримання статусів учнів
    DB-->>Monitor: Повертає дані
    Monitor-->>Dashboard: Відображення учнів
    
    S->>AI: Взаємодія з AI-тютором
    AI->>Monitor: Оновлення активності
    Monitor->>Dashboard: Real-time оновлення
    
    Dashboard-->>T: Показує зміни статусу
    
    alt Учень потребує допомоги
        S->>AI: Запит допомоги
        AI->>Monitor: Створення алерту
        Monitor->>Dashboard: Сповіщення
        Dashboard-->>T: Alert: учень потребує допомоги
        T->>Dashboard: Переглядає діалог
        Dashboard->>DB: Завантаження історії чату
        DB-->>Dashboard: Повертає повідомлення
        T->>Dashboard: Відправляє підказку
        Dashboard->>AI: Передає повідомлення вчителя
        AI-->>S: Показує підказку
    end
    
    T->>Dashboard: Запит аналітики
    Dashboard->>DB: Агрегація даних
    DB-->>Dashboard: Статистика
    Dashboard-->>T: Відображення графіків
\`\`\`

## Діаграма 2.3.4: Система сповіщень та алертів

\`\`\`mermaid
stateDiagram-v2
    [*] --> Monitoring: Система активна
    
    Monitoring --> Normal: Учень працює стабільно
    Monitoring --> NeedsAttention: Виявлено відхилення
    
    Normal --> Monitoring: Продовження моніторингу
    
    NeedsAttention --> LowProgress: Низький прогрес
    NeedsAttention --> Struggling: Багато помилок
    NeedsAttention --> Inactive: Довга неактивність
    NeedsAttention --> HelpRequest: Запит допомоги
    
    LowProgress --> Alert: Генерація сповіщення
    Struggling --> Alert
    Inactive --> Alert
    HelpRequest --> UrgentAlert: Термінове сповіщення
    
    Alert --> TeacherNotified: Вчитель отримує сповіщення
    UrgentAlert --> TeacherNotified
    
    TeacherNotified --> TeacherAction: Вчитель переглядає
    
    TeacherAction --> Intervention: Втручання
    TeacherAction --> Observation: Спостереження
    
    Intervention --> Resolved: Проблема вирішена
    Observation --> Monitoring
    
    Resolved --> Monitoring
\`\`\`

## Діаграма 2.3.6: Експорт та звітність

\`\`\`mermaid
flowchart LR
    Start([Вчитель обирає звіт]) --> Type{Тип звіту}
    
    Type -->|Індивідуальний| Student[Вибір учня]
    Type -->|По класу| Class[Вибір класу]
    Type -->|Порівняльний| Compare[Вибір груп]
    
    Student --> Period1[Період: 03.11-21.11]
    Class --> Period2[Період: 03.11-21.11]
    Compare --> Period3[Період: 03.11-21.11]
    
    Period1 --> Generate1[Генерація даних]
    Period2 --> Generate2[Генерація даних]
    Period3 --> Generate3[Генерація даних]
    
    Generate1 --> Format{Формат}
    Generate2 --> Format
    Generate3 --> Format
    
    Format -->|PDF| PDF[PDF-документ]
    Format -->|Excel| Excel[XLSX-таблиця]
    Format -->|JSON| JSON[JSON-дані]
    
    PDF --> Download[Завантаження]
    Excel --> Download
    JSON --> Download
    
    Download --> End([Звіт отримано])
