# Підрозділ 2.1. Організація навчального середовища здобувача освіти

## Діаграма 2.1.1: Загальна схема взаємодії учня з платформою

\`\`\`mermaid
flowchart TD
    Start([Учень заходить на платформу]) --> Auth{Авторизація}
    Auth -->|Новий користувач| Reg[Реєстрація профілю]
    Auth -->|Є акаунт| Login[Вхід в систему]
    
    Reg --> Profile[Створення учнівського профілю]
    Login --> Dashboard[Головна панель учня]
    Profile --> Dashboard
    
    Dashboard --> Topics[Вибір теми навчання]
    Topics --> AI[Взаємодія з AI-тютором]
    
    AI --> Chat[Діалог з чат-ботом]
    AI --> Tasks[Виконання практичних завдань]
    AI --> Help[Запит допомоги]
    
    Chat --> Adaptive{Адаптивна система}
    Tasks --> Adaptive
    Help --> Adaptive
    
    Adaptive -->|Аналіз прогресу| Level[Визначення рівня складності]
    Level --> Content[Персоналізований контент]
    
    Content --> Progress[Оновлення прогресу]
    Progress --> Badges[Нарахування досягнень]
    Badges --> Stats[Статистика навчання]
    
    Stats --> Continue{Продовжити?}
    Continue -->|Так| Topics
    Continue -->|Ні| Exit([Вихід з платформи])
\`\`\`

## Діаграма 2.1.2: Персоналізована траєкторія навчання

\`\`\`mermaid
graph LR
    A[Початок роботи] --> B[Діагностичне тестування]
    B --> C{Визначення рівня}
    
    C -->|Початковий| D1[Базові концепції]
    C -->|Середній| D2[Поглиблене вивчення]
    C -->|Високий| D3[Складні задачі]
    
    D1 --> E1[Інтерактивні уроки]
    D2 --> E2[Практичні завдання]
    D3 --> E3[Проєктна робота]
    
    E1 --> F[AI-підтримка]
    E2 --> F
    E3 --> F
    
    F --> G{Оцінка результатів}
    G -->|Не засвоєно| H[Додаткові матеріали]
    G -->|Частково| I[Закріплення знань]
    G -->|Повністю| J[Наступна тема]
    
    H --> F
    I --> F
    J --> K[Оновлення профілю учня]
\`\`\`

## Діаграма 2.1.3: Сценарії використання (Use Case)

\`\`\`mermaid
graph TB
    Student([Учень])
    
    Student -->|використовує| UC1[Навчання з AI-тютором]
    Student -->|виконує| UC2[Практичні завдання]
    Student -->|отримує| UC3[Миттєвий зворотний зв'язок]
    Student -->|переглядає| UC4[Особистий прогрес]
    Student -->|заробляє| UC5[Досягнення та бейджі]
    Student -->|запитує| UC6[Допомогу в реальному часі]
    
    UC1 --> AI([AI-система])
    UC2 --> AI
    UC3 --> AI
    UC6 --> Teacher([Вчитель])
    
    AI -->|адаптує| Adapt[Складність контенту]
    AI -->|генерує| Content[Персоналізовані завдання]
    AI -->|аналізує| Progress[Прогрес навчання]
\`\`\`

## Діаграма 2.1.4: Процес взаємодії з AI-тютором

\`\`\`mermaid
sequenceDiagram
    participant U as Учень
    participant UI as Інтерфейс
    participant AI as AI-тютор
    participant DB as База знань
    participant Analytics as Система аналітики
    
    U->>UI: Ставить запитання
    UI->>AI: Передає запит
    AI->>DB: Пошук релевантної інформації
    DB-->>AI: Повертає дані
    AI->>AI: Аналіз контексту діалогу
    AI->>Analytics: Логування взаємодії
    AI-->>UI: Генерує персоналізовану відповідь
    UI-->>U: Відображає відповідь
    
    U->>UI: Виконує завдання
    UI->>AI: Відправляє рішення
    AI->>AI: Перевірка та оцінка
    AI->>Analytics: Оновлення прогресу
    AI-->>UI: Зворотний зв'язок
    UI-->>U: Показує результат та пояснення
