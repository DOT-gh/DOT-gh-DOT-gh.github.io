AI Reality Check

Що насправді вміє AI

  • Розпізнавання образів і тексту
  • Генерація контенту (текст, зображення, код)
  • Аналіз великих обсягів даних
  • Переклад та обробка мови
  • Прогнозування на основі патернів

Чого AI НЕ вміє:

  • Розуміти контекст як людина
  • Мати власну свідомість або емоції
  • Творити щось абсолютно нове
  • Завжди відрізняти правду від вигадки
  • Приймати етичні рішення
  • Pattern matching на стероїдах
  • Статистичні моделі, а не свідомість
  • Навчання на даних → передбачення наступного токена
  • ChatGPT не "розуміє" - він "угадує"
  • Немає здорового глузду, лише вірогідності

IoT Attack Vectors

Як хакери атакують пристрої:

  • Сканування портів (Telnet 23, SSH 22, HTTP 80)
  • Brute-force стандартних паролів
  • Експлуатація вразливостей прошивки
  • Man-in-the-Middle через WiFi перехоплення
  • Фішинг для отримання облікових даних

Ризики для Користувачів

Персональні загрози:

  • Фішинг через email і SMS
  • Зламані IoT камери → витік приватного життя
  • Викрадення банківських даних
  • Ransomware - шифрування файлів за викуп
  • Стеження через смартфони та IoT

Практичний Захист
