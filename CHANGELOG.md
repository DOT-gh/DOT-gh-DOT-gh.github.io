# Changelog

Всі помітні зміни цього проекту будуть задокументовані в цьому файлі.

## [v0.4] - 2025-12-14

### Додано
- Розширена геймфікація з 50+ досягненнями
- Streak freeze система
- Unlock themes для персоналізації
- Funnel chart аналітика
- ML predictions для вчителя
- Детальна аналітика учнів
- Activity heatmap
- Onboarding тур для нових користувачів
- Keyboard shortcuts (Ctrl+K, Ctrl+,, Esc)
- Breadcrumbs навігація
- PWA manifest та service worker
- 404 сторінка з міні-грою
- Changelog

### Покращено
- Code Editor з bracket matching та breakpoints
- Variable inspector в редакторі
- ШІ-тьютор з markdown rendering
- Copy button для code blocks
- Typing indicator "ШІ друкує..."
- Tab = 4 пробіли
- Ctrl+/ для коментування

### Виправлено
- react-syntax-highlighter видалено (проблема завантаження)
- Імпорти компонентів
- Toast notifications працюють коректно
